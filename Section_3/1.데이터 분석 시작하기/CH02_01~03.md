# 2. R studio 사용법

- command + ENTER : 스크립트 실행
- command + D : 해당 라인 삭제하기
- command + SHIFT + c : 주석 처리
- 주석문 + —— 붙이기 : 북마크 표기
- option + - : ← 표기

# 3. 라이브러리 & 기본 명령어 & 객체 이해하기

- setwd(”D:/파일경로”) : 경로에서 데이터 가져오기
- install.packages(”dplyr”) : dplyr 라이브러리 다운로드
- library(dplyr) : dplyr 라이브러리 실행
